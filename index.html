Pipeline Models

Five specialised AI models working in sequence to deliver a complete tumor analysis

Stage 0 — Optional
05

BrainMRDiff

Conditional diffusion model for anatomically consistent synthetic brain MRI generation. Feeds Model 1 with augmented training data.

Under Training
  • DDPM / DDIM sampling
  • TSA structural conditioning
  • AdaGN modality embedding
  • 128×128 slice generation
Stage 1
01

EfficientNetB3

Classifies each 2D MRI slice into one of four categories, then aggregates via majority voting for a patient-level diagnosis.

96.77% Accuracy
  • Glioma / Meningioma / Normal / Pituitary
  • Slice voting across 3D volume
  • Two-stage transfer learning
  • 300×300 input resolution
Stage 2
02

HiResCAM

Generates high-resolution saliency maps showing exactly which regions of the MRI the model focused on to reach its decision.

XAI Explainability
  • Gradient × activation maps
  • Higher resolution than GradCAM
  • JET colormap overlay
  • Top-N confident slices
Stage 3 — Glioma Only
03

3D U-Net

Voxel-level 3D segmentation across all four MRI modalities. Delineates NCR, ED, and ET sub-regions with clinical precision.

98.98% Voxel Accuracy
  • 4-modality input (t1n, t1c, t2w, t2f)
  • 128³ sliding window inference
  • DiceTversky combined loss
  • 3 tumour sub-regions
Stage 4
04

LLM

Synthesises all pipeline outputs into a structured, plain-language clinical report suitable for radiologist review.

Clinical Report
  • Large Language Model
  • Temperature 0.2 — factual output
  • 6-section structured report
  • Runs regardless of tumour type
